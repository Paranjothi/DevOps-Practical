
### Delete Commit History in GitHub:
```
git checkout --orphan newBranch
git add -A 
git commit -m "First Commit"
git branch -D master  
git branch -m master  
git push -f origin master  
git gc --aggressive --prune=all
git branch --set-upstream-to=origin/master master
```
